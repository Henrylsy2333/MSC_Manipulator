NexECM EtherCAT Master

Setup Disk Update History

NexECM is an EtherCAT Master Communication Protocol solution, it is based on IntervalZero RTX 32Bit (RTX is a real-time extension on Micorsoft Windows) to offer real-time communication between EtherCAT Master and EtherCAT slave devices.

Supported Network Interface Support List
  • Intel82574L
  • IntelI210

Update Information
  • VER: version
  • FIX: some issues are fixed in this version
  • MOD: some specifics are modified in this version
  • NEW: some new functions/hardware are supported in this version
